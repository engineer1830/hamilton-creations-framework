🧰 2. Recording Checklist (Professional‑Grade)
Browser Setup
Use Chrome or Edge

Disable extensions

Hide bookmarks bar

Zoom at 100%

Set window size to 1440×900

Use a clean profile if possible

Framework Setup
Ensure you have at least 6–12 cards

Ensure categories have multiple items

Ensure dark mode toggle is visible

Ensure related items exist for the chosen page

Cursor Behavior
Move smoothly

Avoid jitter

Avoid diagonal micro‑movements

Pause briefly before clicks

Don’t move too fast

Environment
Close all other apps

Turn off notifications

Use a neutral desktop background

Record in a quiet environment

Recording Tool
Use one of these:

OBS Studio (recommended)

ScreenFlow

CleanShot X

Snagit

Recording Settings
Resolution: 1440×900

FPS: 30

Format: MP4

Cursor highlight: off

Cursor shadow: off

🎞 3. GIF Editing Workflow
Step 1 — Trim the recording
Use a tool like:

ScreenFlow

Premiere

DaVinci Resolve

Kap (Mac)

EzGIF (web)

Trim to 12–15 seconds.

Step 2 — Speed adjustments
Sidebar interactions: 100% speed

Search filtering: 110% speed

Dark mode toggle: 90% speed (smooth)

Mobile sidebar: 100% speed

Step 3 — Crop
Crop to 16:10 or 16:9 around the main UI.

Avoid showing browser chrome unless you want a “realistic” look.

Step 4 — Color correction
Slight contrast boost (+5%)

Slight saturation boost (+3%)

Sharpen: 0.2–0.3

Step 5 — Add captions
Use a clean, modern style:

Font: Inter SemiBold

Size: 18–22px

Background: subtle translucent black

Padding: 6–10px

Position: bottom center

Duration: 1.5–2 seconds each

Step 6 — Export GIF
Settings:

Width: 1200px

FPS: 20

Dither: Floyd–Steinberg

Colors: 128–256

Loop: infinite

Compression: medium
